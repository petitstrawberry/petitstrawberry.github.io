    Wrapper

    Wrapper(view)

    viewと同じサイズのViewであり、子Viewとしてviewを持つ。viewのCallback FunctionやParameterを変更することなく、新たにCallback FunctionやParameterを設定したい時に使う。

    Sample #

    common def MyView()
    var view = frame(100,100,VStack(\
        Rectangle(#C_RED),\
        Text("MyView"),\
        Rectangle(#C_BLUE)\
    ))
    spvar view,"uikitf_layout","MyView_layout"

    return view
end

common def MyView_layout self
    print "MyView_layout!!!"
end
